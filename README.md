To run this project, run:
```/Minesweeper/Release/Minesweeper.exe```

## Control
- `LMB` - Check the field
- `RMB` - Place the flag
- `R` - Reset
- `Space` - Turn ON / OFF bot
- `Up arrow` - Increase bot click delay
- `Down arrow` - Decrease bot click delay
